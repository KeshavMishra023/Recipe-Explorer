# Recipe App

A modern **Recipe App** built with **React** and **Next.js** that allows users to browse, search, and explore recipes. The app features **pagination**, **search functionality**, and **loading skeletons** for a smooth user experience.

- **Loading skeletons** to improve perceived performance.
- Responsive design for **desktop and mobile devices**.
